# Freelance Marketplace

A full-stack freelancing marketplace where users can explore jobs, post tasks, accept tasks, manage dashboards, and interact through a secure authentication system.

## 🛠 Local Setup
1. Clone the repo:
```sh
git clone https://github.com/Sarwar009/ph-B12A10-FreelanceMarlet-client
```

2. Install dependencies:
```sh
npm install
```

3. Create `.env` file:
```
VITE_API_URL=your_api_url
```

4. Run:
```sh
npm run dev
```
